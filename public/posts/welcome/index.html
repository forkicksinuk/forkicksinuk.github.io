欢迎使用Morandi主题

发布于 2024年01月01日 · 分类: 博客 · 标签: Hugo , 主题 , 莫兰迪

欢迎

欢迎使用 Morandi 主题！这是一个专为 Hugo 设计的优雅博客主题，采用了柔和的莫兰迪配色方案。

什么是莫兰迪色系

莫兰迪色系是以意大利画家乔治·莫兰迪（Giorgio Morandi）命名的一种配色风格。这种色系的特点是：

  • 低饱和度
  • 柔和温暖
  • 优雅宁静
  • 舒适耐看

主题特点

1. 优雅的设计

主题采用简洁的设计语言，让内容成为焦点。每个元素都经过精心设计，确保最佳的阅读体验。

2. 响应式布局

无论是在手机、平板还是桌面电脑上，主题都能完美显示，提供一致的用户体验。

3. 代码高亮

支持代码块显示，适合技术博客：

function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("Morandi");

4. 丰富的排版

支持各种 Markdown 元素，包括：

  • 列表项目
  • 粗体文字
  • 斜体文字
  • 链接

这是一个引用块。莫兰迪色系的美在于它的克制与优雅。

开始使用

  1. 安装 Hugo
  2. 创建新站点
  3. 安装 Morandi 主题
  4. 开始写作

就是这么简单！

结语

希望你喜欢这个主题。如果有任何问题或建议，欢迎
