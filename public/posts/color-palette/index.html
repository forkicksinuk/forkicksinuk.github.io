Morandi主题配色方案

发布于 2024年01月03日 · 分类: 设计 · 标签: 配色 , 主题

配色说明

Morandi 主题采用了精心挑选的莫兰迪色系，每种颜色都经过仔细调配，确保整体的和谐与优雅。

主要色彩

灰粉色 #D4C5C7

  • 用途：按钮、链接悬停、重点强调
  • 特点：温柔而优雅，不失女性柔美

灰褐色 #B8A398

  • 用途：次要文字、元信息
  • 特点：沉稳大气，易于阅读

米白色 #E8E4E1

  • 用途：区块背景、头部底部
  • 特点：纯净舒适，不刺眼

深灰褐 #5C5552

  • 用途：正文文字、标题
  • 特点：清晰易读，对比度适中

灰绿色 #A8B5B2

  • 用途：点缀色、链接
  • 特点：清新自然，活力十足

灰紫色 #B0A8B4

  • 用途：链接、悬停状态
  • 特点：优雅神秘，提升品质感

背景色

奶油白 #F5F2EF

  • 用途：页面主背景
  • 特点：温暖柔和，长时间阅读不疲劳

浅灰 #D5D0CD

  • 用途：边框、分隔线
  • 特点：低调柔和，不喧宾夺主

设计原则

  1. 对比度适中：既保证可读性，又不过于刺眼
  2. 色彩和谐：所有颜色都在同一色调范围内
  3. 层次分明：通过颜色深浅区分信息层级
  4. 优雅耐看：适合长时间阅读和浏览

使用建议

在创作内容时，可以充分利用这些配色：

  • 使用标题和正文的对比来组织内容
  • 适当使用引用块来突出重点
  • 代码块已经过特殊设计，与整体风格协调
  • 图片建议使用同样柔和的色调，以保持整体和谐

自定义配色

如果你想自定义主题配色，可以修改 themes/morandi/static/css/style.css 文件中的 CSS 变量：

:root {
  --morandi-pink: #d4c5c7;
  --morandi-brown: #b8a398;
  --morandi-beige: #e8e4e1;
  /* ... 其他颜色 ... */
}

只需要修改这些变量的值，整个主题的配色就会相应改变。
